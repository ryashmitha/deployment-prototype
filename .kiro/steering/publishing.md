# Publishing Sailwind Prototypes

GitLab Pages publishing for Sailwind prototypes is managed through the centralized ux-sites repo at https://gitlab.appian-stratus.com/docs/ux-sites. Each project gets its own subfolder and URL. The ux-sites repo has its own agent power that guides through the full workflow: building the Vite project, verifying it locally, deploying via a GitLab merge request, and configuring Vite's `base` path for subfolder deployment. Direct users there for any publishing questions.
